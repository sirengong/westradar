📋 如何新增一期报告

1
生成新的报告 HTML 文件，命名为 index.html，封面图片放在同级的 index_files/ 文件夹内
2
在本仓库新建一个以日期命名的文件夹，例如 2026-04-08/，把 index.html 和 index_files/ 放进去
3
打开根目录的 index.html（本页），把示例注释里的卡片复制一份，填入新一期的日期和数据，并把旧的 latest class 去掉
4
把改动 commit & push 到 GitHub，几秒后新报告就自动上线了
